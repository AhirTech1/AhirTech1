I'm a computer science student and tech lead who loves seeing how systems work under the hood. As a heavy Linux user and open-source advocate, I prefer learning by building—whether that's orchestrating multi-agent AI workflows or wiring up custom hardware.

* 🖥️ **Workspace:** Linux Native (CachyOS + Hyprland).
